# PN-002 — Caregivers receiving remote alarms need to acknowledge them so repeated alerts do not disrupt care
A user who has received and is acting on a remote alarm needs to acknowledge the alarm and have the system stop re-issuing the same alarm for a bounded period, so that ongoing care is not disrupted by duplicate notifications while the original condition is being managed.
