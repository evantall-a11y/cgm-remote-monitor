# SUB-001 — Notification dispatcher subsystem
The notification dispatcher subsystem shall accept alert requests submitted by glucose evaluation modules, group requests by alarm group, select the single highest-severity active alert per group within an evaluation cycle, and forward the selected alert onto the server-wide alert distribution channel.
